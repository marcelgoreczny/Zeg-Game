# Zeg-Game
Tytuł : Gra przeglądarkowa - Labirynt.
Opis :Celem projektu jest stworzenie nowoczesnej, przeglądarkowej gry typu labirynt, w której gracz steruje postacią poruszającą się po planszy, zbiera przedmioty, unika zagrożeń oraz rozwiązuje zagadki na kolejnych, coraz trudniejszych etapach.
Czas realizacji : od 10.04.2026 do 08.06.2026.
Autorzy : Marcel Góreczny i Konrad Gawęda.
